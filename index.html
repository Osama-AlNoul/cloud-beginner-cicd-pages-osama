What is CI/CD?

CI/CD is a modern practice in cloud computing that automates the process of integrating code changes and deploying applications. Continuous Integration (CI) ensures that new code is automatically tested and checked. Continuous Deployment (CD) automatically publishes the updated application so users can access the latest version without manual work.

My Pipeline

When I push code to the main branch in GitHub, GitHub Actions automatically runs a workflow that checks if the required files exist. If everything is correct, the workflow deploys the website to GitHub Pages and the website becomes available online.
